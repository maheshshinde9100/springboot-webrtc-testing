# Simple Video Call Demo
## Spring Boot + WebRTC Minimal Implementation

A minimal working video call implementation between two users using Spring Boot backend and WebRTC for peer-to-peer video streaming. Successfully tested and ready for integration into larger projects.

## Status
- Feature Working: Live video and audio call between two browsers
- Core Tech Validated: Spring Boot WebSocket + WebRTC
- Ready for Scaling: Foundation code for bigger projects

## Tech Stack
- Backend: Spring Boot 3.1.5, WebSocket
- Frontend: Vanilla JavaScript, WebRTC API
- Signaling: Custom WebSocket server
